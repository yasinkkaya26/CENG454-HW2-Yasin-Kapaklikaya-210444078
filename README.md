# Sky High Prototype II - Threat Corridor

CENG 454 Game Programming HW2

## Mission Flow
Takeoff → Enter Danger Zone → Missile Launch → Escape → Landing → Mission Complete

## Systems
- Terrain and runway setup
- Danger zone trigger and HUD warning
- Missile launcher and homing
- Aircraft hit response and respawn
- Landing validation and mission completion
- Audio feedback system
